List of HTML tags

  1. HTML- the root element of every html document.
  2. HEAD - contains the data like title,and linked spreeadsheet.
  3. TABLE -defines a table.
  4. TH - defines a table header.
  5. TD- defines a table cell.
  6. HR-gives a horizontal thematic break.
  7. IMG- used to add image in the document.
  8. A-creates hyperlinks to other web pages or resources.
  9. TR- reprents a table row.
  10. SPAN- an inline container used for styling a portion of text.
