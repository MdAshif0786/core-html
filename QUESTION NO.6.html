Full Stack Web Development Tech Stacks

  1. HTML(hyper text markup language)
    2. Primary use case key features/benefits
    Building the structure of web pages
    • simple and easy to learn
    • compatible with all web browsers
    • allows for semantic markup
  2. CSS
    3. primary use case key features/benefits
    • Styling and designing web pages
    • Controlling the layout and appearance of elements
    • Separates content from presentation
    • Offers a wide range of styling options and flexibility
    • Enables responsive design for various screen sizes
  3. JAVASCRIPT
    4. primary use case key features/benefits
    • Adding interactivity to web pages
    • Implementing client-side logic and behavior
    • Allows dynamic content updates without page refresh
    • Supports asynchronous operations (AJAX)
    • Compatible with various web browsers
